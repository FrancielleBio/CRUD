# Cadastro de Clientes - CrudCrud

Projeto simples em HTML, CSS e JavaScript.

## Funcionalidades

- Cadastrar cliente com nome e e-mail
- Listar clientes cadastrados
- Excluir cliente
- Consumo de API com Fetch API
- Requisições GET, POST e DELETE

## Importante

Antes de testar, abra o arquivo `scripts.js` e substitua:

```js
const API_URL = "COLE_AQUI_SUA_URL_DO_CRUDCRUD/clientes";
```

pela sua URL do CrudCrud, por exemplo:

```js
const API_URL = "https://crudcrud.com/api/SEU_CODIGO/clientes";
```
